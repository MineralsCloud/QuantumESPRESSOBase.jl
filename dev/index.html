QuantumESPRESSOBase.titleof — Method
titleof(::Type{<:InputEntry})
titleof(::InputEntry)

Return the title of the input entry in Quantum ESPRESSO.

The definition titleof(x) = titleof(typeof(x)) is provided for convenience so that instances can be passed instead of types.

Examples

julia> titleof(SystemNamelist())
"SYSTEM"

julia> titleof(SystemNamelist)
"SYSTEM"
